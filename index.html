Módulos do Curso

  • O que é Programação?

    Programação é o processo de criar um conjunto de instruções que o computador executa para realizar tarefas. É a linguagem que usamos para “conversar” com máquinas e dizer o que elas devem fazer.

    Variáveis e Tipos de Dados

    Variáveis são “caixas” para guardar informações. Cada variável tem um tipo, como:

    • Number: números (ex: 10, 3.14)
    • String: textos (ex: "Olá Mundo")
    • Boolean: verdadeiro (true) ou falso (false)

    Exemplo de código em JavaScript

    let idade = 14;
let nome = "Ana";
let maiorDeIdade = false;

console.log("Nome:", nome);
console.log("Idade:", idade);
console.log("É maior de idade?", maiorDeIdade);

    Operadores Matemáticos

    Usados para fazer contas no código:

    • + soma
    • - subtração
    • * multiplicação
    • / divisão
    • % resto da divisão

    Exemplo de cálculo simples

    let a = 10;
let b = 3;
let soma = a + b;
let resto = a % b;

console.log("Soma:", soma);
console.log("Resto da divisão:", resto);

  • Introdução ao HTML

    HTML é a linguagem usada para criar a estrutura das páginas web. Com ela, definimos títulos, parágrafos, imagens e links.

    Exemplo de estrutura básica em HTML

    <!DOCTYPE html>
<html>
  <head>
    <title>Minha Página</title>
  </head>
  <body>
    <h1>Olá Mundo</h1>
    <p>Este é um parágrafo.</p>
  </body>
</html>

    Estilizando com CSS

    CSS é usado para deixar a página visualmente bonita, definindo cores, fontes, tamanhos e posicionamentos.

    Exemplo de CSS básico

    body {
  background-color: #121214;
  color: white;
  font-family: Arial, sans-serif;
}

h1 {
  color: #8257e5;
}

  • Introdução ao JavaScript

    JavaScript é uma linguagem que torna as páginas web interativas e dinâmicas, permitindo criar jogos, animações e responder a ações do usuário.

    Funções

    Funções são blocos de código que executam uma tarefa específica. Exemplo:

    function saudacao(nome) {
  console.log("Olá, " + nome + "!");
}

saudacao("Maria");

    Eventos

    Eventos são ações do usuário, como clicar um botão ou mover o mouse, que podem disparar funções.

    document.querySelector("button").addEventListener("click", function() {
  alert("Você clicou no botão!");
});

    Manipulação do DOM

    O DOM é a estrutura da página. Com JavaScript, podemos alterar textos, imagens e estilos da página.

    document.getElementById("demo").innerText = "Texto alterado!";

  • Conceitos de Git

    Git é um sistema que controla versões do seu código, permitindo salvar várias versões e voltar a versões anteriores.

    Comandos básicos

    git init          # Inicia um repositório Git
git add .         # Adiciona arquivos para o próximo commit
git commit -m "Mensagem"  # Salva uma versão com mensagem

    GitHub e Repositórios

    GitHub é uma plataforma online para armazenar seus projetos e colaborar com outros.

    Publicando no GitHub Pages

    Você pode publicar sites estáticos gratuitamente usando o GitHub Pages. Basta enviar seu código para o repositório e ativar a opção.

    Comando para enviar código ao GitHub

    git push origin main

  • Planejamento

    Antes de começar a programar, planeje seu projeto: quais funcionalidades terá, como será o design e o fluxo.

    Desenvolvimento

    Implemente o projeto usando HTML, CSS, JavaScript e outras tecnologias aprendidas.

    Testes e Deploy

    Teste seu projeto para garantir que tudo funcione e publique para que outras pessoas possam acessar.

  • Funções Avançadas

    Aprenda funções de ordem superior e closures.

    // Função que retorna outra função
function saudacaoPersonalizada(nome) {
  return function() {
    console.log("Olá, " + nome + "!");
  };
}

const olaAna = saudacaoPersonalizada("Ana");
olaAna(); // Olá, Ana!

    Arrow Functions

    const soma = (a, b) => a + b;
console.log(soma(3, 4)); // 7

  • Selecionar Elementos

    Usando querySelector e querySelectorAll.

    const btn = document.querySelector("button");
const itens = document.querySelectorAll(".item");

    Criar e Remover Elementos

    // Criar um novo parágrafo
const p = document.createElement("p");
p.textContent = "Novo parágrafo";
document.body.appendChild(p);

// Remover um elemento
const elemento = document.getElementById("remover");
elemento.remove();

  • O que são APIs?

    APIs permitem que seu programa se comunique com outros serviços.

    Usando fetch para consumir API

    fetch("https://api.exemplo.com/dados")
  .then(response => response.json())
  .then(data => {
    console.log(data);
  })
  .catch(error => console.error(error));

  • O que é Backend?

    Parte do site ou app que roda no servidor e cuida de dados, segurança e regras.

    Exemplo básico com Node.js

    const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Olá, mundo do backend!');
});

server.listen(3000, () => {
  console.log('Servidor rodando na porta 3000');
});

  • Objetivo

    Crie um projeto completo que envolva frontend, backend e consumo de APIs.

    Dicas para o projeto

    • Planeje funcionalidades
    • Use Git para controle de versão
    • Implemente interatividade no frontend
    • Consuma APIs externas
    • Hospede seu projeto online (GitHub Pages, Heroku, etc.)
