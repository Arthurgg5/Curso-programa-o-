Módulos do Curso

  • O que é Programação?

    Programação é o processo de criar um conjunto de instruções que dizem ao computador o que fazer.

    Variáveis e Tipos de Dados

    Variáveis são como caixas que guardam informações. Exemplos de tipos:

    • Number: Números, como 10, 3.14
    • String: Texto, como "Olá Mundo"
    • Boolean: Verdadeiro (true) ou falso (false)

    Exemplo de código em JavaScript

    
let idade = 14;
let nome = "Ana";
let maiorDeIdade = false;

console.log("Nome:", nome);
console.log("Idade:", idade);
console.log("É maior de idade?", maiorDeIdade);

    Operadores Matemáticos

    Usados para fazer contas:

    • + (soma)
    • - (subtração)
    • * (multiplicação)
    • / (divisão)
    • % (resto da divisão)

    Exemplo de cálculo simples

    
let a = 10;
let b = 3;
let soma = a + b;
let resto = a % b;

console.log("Soma:", soma);
console.log("Resto da divisão:", resto);

  • Introdução ao HTML

    HTML é a linguagem que estrutura as páginas web.

    
<!DOCTYPE html>
<html>
  <head>
    <title>Minha Página</title>
  </head>
  <body>
    <h1>Olá Mundo</h1>
    <p>Este é um parágrafo.</p>
  </body>
</html>

    Estilizando com CSS

    CSS deixa sua página bonita e estilizada.

    
body {
  background-color: #121214;
  color: white;
  font-family: Arial, sans-serif;
}

h1 {
  color: #8257e5;
}

  • Introdução ao JavaScript

    JavaScript deixa a página interativa e dinâmica.

    Funções

    Funções são blocos de código que executam uma tarefa.

    
function saudacao(nome) {
  console.log("Olá, " + nome + "!");
}

saudacao("Maria");

    Eventos

    Eventos são ações que o usuário faz, como clicar um botão.

    
document.querySelector("button").addEventListener("click", function() {
  alert("Você clicou no botão!");
});

    Manipulação do DOM

    Você pode alterar o conteúdo da página usando JavaScript.

    
document.getElementById("demo").innerText = "Texto alterado!";

  • Conceitos de Git

    Git é um sistema de controle de versões para seu código.

    
git init
git add .
git commit -m "Primeiro commit"

    GitHub e Repositórios

    GitHub é uma plataforma para armazenar e compartilhar seu código.

    Publicando no GitHub Pages

    Você pode publicar sites gratuitamente no GitHub.

    
git push origin main
