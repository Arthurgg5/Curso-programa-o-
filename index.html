Módulos do Curso

  • Funções Avançadas

    Funções de ordem superior são funções que podem receber ou retornar outras funções.

    
function saudacaoPersonalizada(nome) {
  return function() {
    console.log(`Olá, ${nome}! Seja bem-vindo(a).`);
  };
}

const olaAna = saudacaoPersonalizada("Ana");
olaAna(); // Saída: Olá, Ana! Seja bem-vindo(a).

    Arrow Functions

    Arrow Functions são formas mais modernas e compactas de escrever funções.

    
// Versão tradicional
function somaTradicional(a, b) {
  return a + b;
}

// Versão com arrow function
const soma = (a, b) => a + b;
console.log(soma(3, 4)); // Saída: 7

  • Selecionar Elementos

    Selecionar elementos HTML é essencial para interações com JavaScript.

    
const btn = document.querySelector("button");
const itens = document.querySelectorAll(".item");

    Criar e Remover Elementos

    Você pode adicionar ou remover elementos dinamicamente no DOM.

    
// Criar novo parágrafo
const p = document.createElement("p");
p.textContent = "Parágrafo criado com JS";
document.body.appendChild(p);

// Remover elemento existente
const remover = document.getElementById("remover");
if (remover) remover.remove();

  • O que são APIs?

    API é uma ponte entre diferentes sistemas. Com JavaScript, usamos principalmente via HTTP.

    Usando Fetch

    
fetch("https://api.exemplo.com/dados")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Erro na API:", error));

  • O que é Backend?

    Backend é a parte invisível de um site, responsável por regras, dados e segurança.

    Servidor simples com Node.js

    
const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Olá do backend!');
});

server.listen(3000, () => {
  console.log('Servidor rodando na porta 3000');
});

    Checklist do Projeto:

    • Estrutura HTML semântica
