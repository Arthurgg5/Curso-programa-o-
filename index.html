Módulos do Curso

  • O que é Programação?

    Programação é o processo de criar um conjunto de instruções que dizem ao computador o que fazer.

    Variáveis e Tipos de Dados

    Variáveis são como caixas que guardam informações. Exemplos de tipos:

    • Number: Números, como 10, 3.14
    • String: Texto, como "Olá Mundo"
    • Boolean: Verdadeiro (true) ou falso (false)

    Exemplo de código em JavaScript

    
let idade = 14;
let nome = "Ana";
let maiorDeIdade = false;

console.log("Nome:", nome);
console.log("Idade:", idade);
console.log("É maior de idade?", maiorDeIdade);

  • O que são APIs?

    APIs permitem que seu programa se comunique com serviços externos.

    Usando Fetch

    
fetch("https://api.exemplo.com/dados")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

  • Servidor básico com Node.js

    
const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Olá do backend!');
});

server.listen(3000, () => {
  console.log('Servidor rodando!');
});
